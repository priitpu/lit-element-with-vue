# Primaarsed nupud

 Primaarne nupp peaks alati tähistama tegevusvoo soovitud tulemust.

 N: otsingu puhul otsingu päringu tegemine, andmete sisestamisel salvestamine jne. Nupu nimetused peavad arusaadavalt kommunikeerima tegevust, mida kasutajad saavad teha.
 Nimetused peavad olema võimalikult lühikesed ja konkreetsed. Nuppude puhul oleme ära lahendanud enam levinud teksti ja ikooni kombinatsioonid

<Preview>
  <Story name="with label">
    {() => html`
      <t-button>Label</t-button>
    `}
  </Story>
  <Story name="with label and left icon">
    {() => html`
      <t-button>
        <t-icon slot="l-icon" glyph="info"></t-icon>
        Label
      </t-button>
    `}
  </Story>
  <Story name="with label and right icon">
    {() => html`
      <t-button>
        Label
        <t-icon slot="r-icon" glyph="arrow_drop_down"></t-icon>
      </t-button>
    `}
  </Story>
  <Story name="with label and both icons">
    {() => html`
      <t-button>
        <t-icon slot="l-icon" glyph="info"></t-icon>
        Label
        <t-icon slot="r-icon" glyph="arrow_drop_down"></t-icon>
      </t-button>
    `}
  </Story>
  <Story name="icon without label">
    {() => html`
      <t-button iconOnly="true">
        <t-icon glyph="info"></t-icon>
      </t-button>

  <Story name="icon with label">
    {() => html`
      <t-button>Label</t-button>
    `}
  </Story>
</Preview>

Vaikimisi on kasutusel keskmise (32px) kõrgusega nupud. Allpool on ära toodud võimalikud alternatiivid, mida vastavalt vajadusele kasutada.
Nuppude suurused laienevad kõikidele nuppude tüüpidele. Väiksemad nupud on mõeldud kasutamiseks mahukates vaadetes, kus on palju informatsiooni. N: tabeli rea lõpus.
